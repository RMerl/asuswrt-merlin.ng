Update Software

Step 1: Obtain an updated software image file.

Step 2: Enter the path to the image file location in the box below or click the "Browse" button to locate the image file.

Step 3: Click the "Update Software" button once to upload the new image file.

NOTE: The update process takes about 2 minutes to complete, and your Router will reboot.

Allow image update to repartition eMMC if required.
(Note: This option may destroy all data on eMMC. Should only be enabled on a board with a cfeonly image)
