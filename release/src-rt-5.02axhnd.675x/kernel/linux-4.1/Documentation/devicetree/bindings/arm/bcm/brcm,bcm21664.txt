Broadcom BCM21664 device tree bindings
--------------------------------------

This document describes the device tree bindings for boards with the BCM21664
SoC.

Required root node property:
  - compatible: brcm,bcm21664

Example:
	/ {
		model = "BCM21664 SoC";
		compatible = "brcm,bcm21664";
		[...]
	}
