AK4642 I2C transmitter

This device supports I2C mode only.

Required properties:

  - compatible : "asahi-kasei,ak4642" or "asahi-kasei,ak4643" or "asahi-kasei,ak4648"
  - reg : The chip select number on the I2C bus

Example:

&i2c {
	ak4648: ak4648@0x12 {
		compatible = "asahi-kasei,ak4642";
		reg = <0x12>;
	};
};
