Broadcom MDK Programmer's Reference

1. Introduction

The Broadcom Mini Driver Kit (MDK) is a basic programming interface for the Broadcom network switching chips.

The MDK has been designed to support the following types of applications:

  • Bootloaders
  • Unmanaged L2 initialization
  • Static/simple configurations
  • System bringup, debug, and testing
  • Simple custom feature and driver development
  • General applications with low system resources
The MDK does not require interrupts or an operating system to work and basic driver support usually takes up less than 64 KB of memory space.

The MDK does not contain nor is it designed to replace the full feature set and functionality of the Broadcom Software Development Kit (SDK) for Broadcom devices.
