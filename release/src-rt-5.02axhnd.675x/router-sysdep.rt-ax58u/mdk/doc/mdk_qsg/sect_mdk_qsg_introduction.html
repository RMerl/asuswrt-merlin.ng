Broadcom MDK Quick Start Guide

1. Introduction

This document is the "Broadcom MDK Quick Start Guide". If you are new to the MDK (Mini Driver Kit) this is a good place to start getting familiar with the MDK package.

The MDK is intended as a supplement to the standard Broadcom SDK (Switch Development Kit) and compared to the SDK the MDK provides the following major features:

  • Small memory footprint
  • No operating system support required
  • No interrupt support required
  • Scalable architecture
The MDK Quick Start Guide will introduce the main components of the MDK and help you build your first application based on the MDK.
