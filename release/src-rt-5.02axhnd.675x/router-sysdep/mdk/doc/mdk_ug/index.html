8. BMD API Function Reference

Alphabetical list of all APIs

bmd_attach  Attach BMD device to a CDK device.
bmd_cpu_mac_addr_add  Configure CPU MAC address.
bmd_cpu_mac_addr_remove  Delete CPU MAC address.
bmd_detach  Detaches BMD device from a CDK device.
bmd_download  Download code to subdevice.
bmd_init  Initialize switch chip.
bmd_port_mac_addr_add  Configure port MAC address.
bmd_port_mac_addr_remove  Delete port MAC address.
bmd_port_mode_get  Get current port mode.
bmd_port_mode_set  Set port mode.
bmd_port_mode_update  Update port mode based on link status.
bmd_port_stp_get  Get spanning tree protocol state.
bmd_port_stp_set  Set spanning tree protocol state.
bmd_port_vlan_get  Get default VLAN for a port.
bmd_port_vlan_set  Set default VLAN for a port.
bmd_reset  Reset switch chip.
bmd_rx_poll  Poll for Rx packet complete.
bmd_rx_start  Submit Rx packet to DMA queue.
bmd_rx_stop  Abort Rx DMA.
bmd_stat_clear  Clear statistics counter.
bmd_stat_get  Get statistics counter.
bmd_switching_init  Initialize chip for L2 switching.
bmd_test_interrupt_assert  Assert test interrupt.
bmd_test_interrupt_clear  Clear test interrupt.
bmd_tx  Transmit a packet.
bmd_vlan_create  Create a VLAN.
bmd_vlan_destroy  Destroy VLAN.
bmd_vlan_port_add  Add port to a VLAN.
bmd_vlan_port_get  Get list of ports belonging to VLAN.
bmd_vlan_port_remove  Remove port from a VLAN.
