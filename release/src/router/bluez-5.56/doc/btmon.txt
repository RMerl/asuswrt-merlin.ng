BTMON(1)
========
:doctype: manpage


NAME
----
btmon - Bluetooth monitor


SYNOPSIS
--------
*btmon* ['OPTIONS']


DESCRIPTION
-----------
The btmon(1) command provides access to the Bluetooth subsystem monitor
infrastructure for reading HCI traces.


AUTHOR
------
