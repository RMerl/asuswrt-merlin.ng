AiMesh
<#Description#>
將另外一台華碩路由器設定為 AiMesh router，只需要您將它更新到支援 AiMesh 的韌體，即可建立您的 AiMesh system 以提升網路覆蓋率。
AiMesh router 在 AiMesh system 中扮演管理者的角色，您可以在 AiMesh router 的管理介面上，調整 AiMesh system 的所有網路設定。
包括 WiFi 網路名稱 (SSID)，或是登錄新裝置到 AiMesh system 中。
因此需要您連接到支援 AiMesh 的華碩路由器上，我們將自動為您引導到新增裝置的設定畫面。
